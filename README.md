# Приложение «Финансы»

**Персональный финансовый трекер с аналитикой**

Удобное десктопное приложение для учёта доходов и расходов с мощной аналитикой и визуализацией.

## Возможности

- Добавление доходов и расходов с категориями
- Система категорий с цветовой маркировкой
- Аналитика на базе **Pandas**:
  - Расходы по категориям за период
  - Сравнение доходов и расходов
  - Финансовая сводка (баланс)
- Визуализация данных:
  - Круговая диаграмма расходов
  - Столбчатая диаграмма «Доходы vs Расходы»
- Генерация персонализированных советов по экономии
- Сохранение данных в JSON (поддержка нескольких пользователей)

## Технологии

- **Python 3**
- **Tkinter**
- **Pandas** — анализ данных
- **Matplotlib** — построение графиков внутри приложения
- JSON для хранения данных
